# Signal-Classification-Using-deep-learning
This project focuses on classifying different radio signal types using machine learning techniques. Signals such as FM broadcast radio, AM broadcast radio, and Airband voice signals were collected using RTL-SDR hardware and processed using GNU Radio.

The collected signals were converted into IQ samples, from which multiple features were extracted in the time domain, frequency domain, and statistical domain. These features were then used to train machine learning models for automatic signal type classification.

The project demonstrates how signal processing and machine learning can be combined to analyze and identify wireless signals.
